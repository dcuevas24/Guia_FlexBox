Display

Con la propiedad display: flex definimos el contenedor flexible. Esta es la primera propiedad que debemos utilizar para trabajar con el módulo de Flexbox.

flex-direction

Cambia la orientación del eje principal, es decir define la dirección que tienen los ítems. Esta dirección puede ser horizontal o vertical.

flex-direction: row

Es el valor por defecto. Define la dirección horizontal para el eje principal y acomoda los ítems de izquierda a derecha.

CSS

.container {
display: flex;
flex-direction: row;
}
