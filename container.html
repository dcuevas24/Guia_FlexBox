Display

Con la propiedad display: flex definimos el contenedor flexible. Esta es la primera propiedad que debemos utilizar para trabajar con el módulo de Flexbox.

flex-direction

Cambia la orientación del eje principal, es decir define la dirección que tienen los ítems. Esta dirección puede ser horizontal o vertical.

flex-direction: row

Es el valor por defecto. Define la dirección horizontal para el eje principal y acomoda los ítems de izquierda a derecha. 

      
      .container {
        display: flex;
        flex-direction: row;
      }

flex-wrap

Esta propiedad evita el desbordamiento de los ítems es decir establece si los ítems estarán en una sola línea o varias líneas.

flex-wrap: nowrap

Es el valor por defecto. Todos los ítems estarán a lo largo de una sola línea. Notamos que en el ejemplo los items cambian su tamaño. 

      
      .container {
        display: flex;
        flex-warp: nowrap;
      }

justify-content

Esta propiedad define el alineamiento de los ítems a lo largo del eje principal, así como el espacio que existe entre ellos.

justify-content: flex-start

Es el valor por defecto. Agrupa los ítems al inicio del eje principal. 

      
      .container {
        display: flex;
        justify-content: flex-start;
      }
